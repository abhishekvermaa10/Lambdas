<h1 align="center">Lambda Expressions</h1>

<p align="justify">
    Welcome to the <b>Lambda Expressions</b> repository! This repository provides hands-on demos illustrating how
    <b>Lambda Expressions</b> enhance code readability and conciseness by replacing anonymous classes. It explores
    functional interfaces, method references, and exception handling within lambdas, making Java programming more
    expressive and efficient.
</p>

<hr>

<h3 align="left">üìå Table of Contents</h3>
<ol align="left">
    <li>Creating Lambda Expression for method which accepts single argument and return a datatype and replacing
        it with Function</li>
    <li>Creating Lambda Expression for method which accepts single argument and return a boolean and replacing
        it with Predicate</li>
    <li>Creating Lambda Expression for method which accepts single argument and return a void and replacing it
        with Consumer</li>
    <li>Creating Lambda Expression for method which accepts nothing and return a datatype and replacing it with
        Supplier</li>
    <li>Creating Lambda Expression for method which accepts double arguments and return a datatype and replacing
        it with BiFunction</li>
    <li>Creating Lambda Expression for method which accepts double arguments and return a boolean and replacing
        it with BiPredicate</li>
    <li>Creating Lambda Expression for method which accepts double arguments and return a void and replacing it
        with BiConsumer</li>
    <li>Throwing unchecked and checked exception from Lambda Expressions</li>
    <li>Understanding Static Method Reference</li>
    <li>Understanding Instance Method Reference of a particular object</li>
    <li>Understanding Instance Method Reference of an arbitrary object of a particular type</li>
    <li>Understanding Constructor Reference</li>
</ol>
